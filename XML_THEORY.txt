XML = extensible Markup Language

In XML you can use your own tags, tags are not Limited and not predefined.
In XML we use tag or marks.

<!DOCTYPE html>

<?xml version="1.0" encoding="UTF-8"?>


XML was designed to store and transport data.
XML was designed to be both human and machine readable.

Rules for XML=>
1> There is only one root Tag.
2> xml is case sensitive.
3> every opening tag needs a closing tag.
4> Every attribute value is inside double quote.

Two Types of XML =>
1> Structured  (DTD - Document Type Definition)
2> Content (XSD - XML Schema Definition)

Structure of elements in XML Documents:

<root> can be called Parent Element
		<child>
				<subchild> ---- </subchild>
		</child>
</root>
